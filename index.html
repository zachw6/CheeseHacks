What is CheeseHacks?

A student run Hackathon to gain development experience in building web applications to address real-world issues, such as healthcare, return-to-classroom.

Workshops

Wireframing Tutorial

Tuesday - Oct. 12, 2021, 7-8PM

Discord

Join Owen in the WebDev discord to learn more about what wireframing is and how it can be useful to map out a plan for your web app. We’ll show off a couple free options through Figmas and Adobe XD!

HTML/CSS Tutorial

Thursday - Oct. 14, 2021 7-8PM

Join Bailey to learn the basics of website building, through a crash course in HTML and CSS. We’ll show you how to build a basic webpage, and if there’s some time at the end, deploy it to the internet!

Javascript/Node/npm Tutorial

Tuesday - Oct. 19, 2021 7-8PM

Join Hunter in the WebDev discord to learn more about the powerful web language that is javascript! We’ll talk about how you can take your coding to the next level by importing node packages with the Node Package Manager, and show some fun package examples!


Git/Github Tutorial

Thursday - Oct. 21, 7-8pm

Join Hunter in the WebDev discord to learn more about the powerful web language that is javascript! We’ll talk about how you can take your coding to the next level by importing node packages with the Node Package Manager, and show some fun package examples!
