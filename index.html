/ * What is CheeseHacks? * /

CheeseHacks is a beginner friendly fun 24-hour Hackathon hosted by GDSC UW Madison and the Web Dev Club on the last weekend of October.

We'll be hosting a series of helpful workshop leading up to the Hackathon weekend starting last week of September to help prepare for the hackathon.

/ * Workshops * /

Introduction to Wireframing

Tuesday - Oct. 12, 2021

Join Bailey (President of UW-Madison Web Development Club) for an evening of learning about how to use HTML and CSS concepts. Recommended for beginners and those getting started with making their first site.

Introduction to HTML/CSS

Thursday - Oct. 14, 2021

Sit down with Owen (Secretary of UW-Madison Web Development Club) to learn more about how to get started with application design, from web to mobile!
